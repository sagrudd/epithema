# Family-to-Tool Mapping Reference

## 1. Status

This appendix is a maintained reference location for the EMBOSS-RS family-to-tool
mapping. It supports governance and release review but does not supersede the
normative rules in the governance manual or policy documents.

The canonical governance context for this appendix is
[EMBOSS-RS Governance Manual](../emboss_rs_governance_manual.md).

## 2. Registry Use

This file should be updated when new governed tool families or governed tools are
introduced. It exists to provide one durable location for the mapping between
family concepts and the concrete `emboss-rs <tool> ...` surface.

## 3. Initial Registry

The detailed family registry has not yet been populated in this repository
checkout. Until that registry is established through future planning or delivery
work, the governing assumptions are:

- the public execution surface is `emboss-rs <tool> ...`
- governed tools should be grouped and tracked by family
- plotting-oriented workflows belong in `emboss-r`

## 4. Future Expansion

When the tool inventory becomes concrete, this appendix should record at least:

- family name
- governed tool name
- implementation status
- documentation status
- relevant notes on `emboss-r` integration where applicable
